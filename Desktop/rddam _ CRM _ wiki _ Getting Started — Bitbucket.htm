Overview

This document helps to setup a development environment to start working on the CRM project.

Setting-up Development Environment

0. Pre-requisites

  • git
  • Access to the bitbucket repository: https://bitbucket.org/rddam/crm
  • XAMPP (which includes PHP >= 5.4, MCrypt, MySQL)
  • Make sure your PATH is set to use the PHP from XAMPP rather than a local PHP.
  • Composer
  • A local folder to store the source codes (let's called it SRC)
  • PEAR

  • PHING

  • Optional: Apache with mod_rewrite enabled (instead of this, we could use the PHP built-in development server)

1. Get the source code

  • From folder SRC: run git clone command (replace username with your Bitbucket username): git clone https://username@bitbucket.org/rddam/crm.git

2. Install dependencies using Composer

  • From folder SRC, run following command

  • Windows environment: composer update

  • Linux environment: php composer.phar update

3. Setup Database

  • Create 2 databases: rddam_crm (for development) and rddam_crm_test (for testing). Use root / empty password.
  • In case you need to configure a different database or passwords, update /app/config/database.php and /app/config/testing/database.php accordingly:
'mysql' => array(
    'driver'    => 'mysql',
    'host'      => 'localhost',
    'database'  => '<database_name>',
    'username'  => '<username>',
    'password'  => '<password>',
    'charset'   => 'utf8',
    'collation' => 'utf8_unicode_ci',
    'prefix'    => ''
)
  • From folder SRC: run Lavarel artisan db commands.
    • php artisan migrate This will create all common database tables.
    • php artisan migrate --path app/database/migrations/crm This will create all CRM database tables.
    • php artisan db:seed This will populate some simple data (or seed the data).

4. Start the application

  • From folder SRC: php artisan serve to start the PHP built-in development server
  • The application can be accessed from: http://localhost:8000

5. Optional: Run test

  • Seed the test database - from folder SRC: php artisan migrate --seed --env="testing"

Optional: Other Information

Install Laravel PHP framework

This is not required, but for other new projects, you will need to install Laravel in order to get started.

  • From folder SRC, run composer command: composer create-project laravel/laravel --prefer-dist

Install Codeception

This is not required, but for other new projects, you will need to install Codeception in order to get started.

  • From folder SRC, run composer command: composer require "codeception/codeception:*"
